# Shipment Sure Delivery Prediction

**Author:** Thummala Thanus Thapasvi  
**Live App:** https://shipmentsure-thapasvi.streamlit.app/  
**Model:** XGBoost  
**Tech:** Python, Streamlit, scikit-learn, XGBoost  

**Description:** Predicts if a shipment will be **on time** or **delayed** using logistics data.

### Features
- Real-time prediction
- Confidence score
- Clean UI
- Deployed using GitHub and Streamlit

### Files
- `app.py` → Web interface
- `requirements.txt` → Dependencies
- `shipment_best_model.pkl` → Model
- `feature_order.pkl` → Feature order
- `scaler.joblib` → scaler
